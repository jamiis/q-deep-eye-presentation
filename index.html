Bias-Variance Tradeoff


Network Training

A light weight master node signals multiple slaves to each train a different network architecture Each slave is a g2.xlarge GPU-focused AWS machine.

Running the Ensemble

On each iteration the master node dispatches the 4 frames to each slave node, or each trained network. All networks return their opinion for the best command back to the master. The master will then averages among these options and start the process over.
